HTML adalah sebuat scripting yang digunakan untuk membuat struktur pada halaman application web. Dengan HTML developer dapat membuat sebuah kerangka sebuah aplikasi website

Selain itu, setelah membuat sebuah halman website dengan CSS kita dapat memperindah halaman sehingga halaman terlihat lebih menarik.

Dan apabila developer ingin menjadikan halaman nya menjadi lebih menarik, developer dapat menggunakan javascript sebagai script yang digunkan untuk memperkaya tampilan pada website, seperti

      Menambahkan aplikasi
      Membuat website lebih dynamic
      Membuat request client dan server lebih ringan

Itulah salah satu cara pengembangan sebuah website
